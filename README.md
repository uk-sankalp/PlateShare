# PlateShare 🟢

PlateShare is a community-first platform built to reduce food waste and ensure surplus food reaches those who truly need it. It brings together donors, volunteers, and NGOs into a single, coordinated system where every contribution makes a difference.

---

## 📌 Features

- **Role-based dashboards**  
  Tailored experiences for Donors, NGOs, Volunteers, and Admins.

- **Instant notifications**  
  Stay updated when food is available nearby for quicker response.

- **Smart matching system**  
  Efficiently connects donors with the right people for distribution.

- **Seamless communication**  
  Built-in chat for smooth coordination between all participants.

- **Delivery lifecycle tracking**  
  Monitor every step—from donation to pickup.

- **Location-based discovery**  
  Find and share food using proximity-based search.

- **Feedback & ratings**  
  Build trust through transparent reviews and ratings.

- **Clean and responsive UI**  
  Modern interface with dark mode and smooth user experience.

---

## 📂 Project Structure

    /frontend   → React (Vite) + Tailwind CSS  
    /backend    → Node.js + Express + MongoDB  

---

## ⚙️ Getting Started

### Backend Setup

    cd backend
    npm install
    npm start

### Frontend Setup

    cd frontend
    npm install
    npm run dev

---

## 📌 Tech Stack

    Frontend        → React 18, Vite, Tailwind CSS  
    Backend         → Node.js, Express, MongoDB (Mongoose)  
    Authentication  → JWT  
    Security        → BCrypt  
    Real-time       → Notification system (polling / webhook-ready)

## 🌱 Vision

To make food sharing simple, fast, and impactful—so that no good food goes to waste.
