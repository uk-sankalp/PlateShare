# PlateShare 🍲

PlateShare is a community-driven platform designed to reduce food waste and help those in need by connecting food donors directly with volunteers and NGOs.

## 🚀 Key Features

- **Multi-Role Support**: Custom dashboards for **Donors**, **NGOs**, **Volunteers**, and **Admins**.
- **Real-time Notifications**: Instant alerts when food is donated nearby, keeping the community connected.
- **Smart Food Matching**: Connects donors with those who can claim and distribute food efficiently.
- **Live Chat System**: Integrated messaging for seamless coordination between donors and claimants.
- **Delivery Tracking**: Manage the end-to-end journey of food donations from posting to pickup.
- **Location Awareness**: Built-in location picker and distance-based searching for proximity matching.
- **Rating & Feedback**: Trust-building system with ratings for all participants in the food cycle.
- **Modern UI/UX**: Dark mode support, responsive design, and smooth animations using Tailwind CSS.

## 📁 Project Structure

- **`/frontend`**: The React-based user interface, powered by Vite and Tailwind CSS.
- **`/backend`**: The robust Node.js and Express API server with MongoDB integration.

## 🛠️ Quick Start

### 1. Backend Setup
```bash
cd backend
npm install
npm start
```

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

## 🌐 Tech Stack
- **Frontend**: React 18, Vite, Tailwind CSS, Lucide Icons.
- **Backend**: Node.js, Express, Mongoose (MongoDB).
- **Security**: JWT (JSON Web Tokens) for authentication, BCrypt for password hashing.
- **Real-time**: Custom notification context and polling/webhook ready system.
